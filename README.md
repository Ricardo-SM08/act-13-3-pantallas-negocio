# MySelftCar

## PROMPT

Mi proyecto tiene la siguiente estructura de carpetas: el archivo main.dart está en la raíz de lib/, y las pantallas están dentro de una carpeta llamada lib/pantallas/.

Necesito que generes el código completo y estético para una nueva pantalla llamada carrito_screen.dart con las siguientes características:

1. Estructura Base:

AppBar: Fondo color lila (Color(0xFFDCD0FF)), título 'MySelfCar' a la izquierda en cursiva negra, y un menú de hamburguesa negro a la derecha.

BottomNavigationBar: Mismo fondo lila, con 3 ítems (Casa, Carrito, Buscar). El ítem 'Carrito' debe estar seleccionado.

2. Diseño del Body (Scrollable):

Banner Superior: Un contenedor rectangular color morado oscuro con la palabra 'CARRITO' en letras blancas centradas. Este contenedor debe tener un borde y una sombra exterior que simulen un brillo de luz neón.

Tarjeta de Producto: Debajo del banner, un contenedor principal con un delineado color morado claro y un efecto neón suave. Dentro de este contenedor debe ir una fila (Row):

Lado Izquierdo: Un contenedor interno que muestre la imagen de un producto (usa una URL de red de ejemplo).

Lado Derecho: La descripción del producto en forma de lista. Cada viñeta de la lista debe ser un icono de estrella.

Esquina Inferior Derecha: Un pequeño botón de icono con una 'X' (color rojo) posicionado en la esquina para simular la eliminación del producto.

Botón de Compra: Debajo de la tarjeta del producto, dejando un buen margen, coloca un botón ancho que diga 'COMPRAR'. Debe usar colores morados que combinen con la paleta y tener también un estilo neón llamativo.

3. REGLA TÉCNICA ESTRICTA:

Para cualquier sombra, transparencia o color, ESTÁ PROHIBIDO usar .withOpacity() ya que está obsoleto en mi versión de Flutter. Debes usar exclusivamente .withAlpha() con valores enteros de 0 a 255 (por ejemplo, Colors.black.withAlpha(26)).

Asegúrate de que el código sea limpio, moderno y la interfaz se vea muy atractiva y coherente."
